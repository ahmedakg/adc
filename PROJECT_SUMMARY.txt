QUICK START INSTRUCTIONS
================================================================

1. Extract Package:
   tar -xzf adc-complete-v2.tar.gz
   cd adc-project

2. Install Dependencies:
   npm install

3. Run Development Server:
   npm run dev
   Open: http://localhost:3000

4. Build for Production:
   npm run build
   npm start

5. Deploy to Vercel:
   bash deploy.sh
   OR
   vercel --prod
